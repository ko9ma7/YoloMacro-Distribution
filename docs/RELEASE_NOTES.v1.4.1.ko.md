# YoloMacro v1.4.1 릴리즈 노트

배포일: 2026-07-22

## 시작 화면

- 유효한 서명 활성화에서는 프로그램 시작 시 **RPA 실행** 탭을 엽니다.
- 최종 비활성·만료·서명 오류·네트워크/응답 오류·최소 버전 미달 상태에서만 **설정**으로 이동합니다.
- 활성화 확인 중 임시 상태는 화면을 강제로 바꾸지 않습니다.
- 설정의 수동 `다시 확인` 또는 업데이트 확인 성공 시 현재 탭을 유지합니다.

## 문서와 배포

- 한국어·영문 README, 사용자 설명서와 활성화 운영 가이드를 공개 Pages에 게시했습니다.
- 소스와 구자료는 Private 저장소에 보존하고, 검증된 ZIP·DLL·SBOM·manifest만 공개 Distribution에 배포합니다.

## 호환성

- 기존 1.4.0 프로젝트와 설정은 그대로 사용할 수 있습니다.
- 공개 서명 manifest는 1.4.0을 최소 호환 버전으로 유지하고 1.4.1을 최신 버전으로 안내합니다.
