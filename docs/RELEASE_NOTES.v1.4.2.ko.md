# YoloMacro v1.4.2 릴리즈 노트

배포일: 2026-07-23

## 폴더와 목록의 순차 실행

- 액션 성공 뒤 새 화면을 캡처해도 같은 폴더의 남은 하위 동작이나 다음 상위 작업을 건너뛰지 않습니다.
- 상위 목록과 조건 폴더 내부를 화면에 보이는 순서대로 계속하고, 목록 끝에서만 0번 항목으로 돌아갑니다.
- 실행 로그에 캡처 정책, 입력 경로, 동작 전/후 대기와 다음 검사 항목을 표시합니다.

## 고속 클릭과 키 입력

- 한 번 발견한 위치를 1~10,000회 클릭하는 `고속 반복 좌클릭`과 0~1,000ms 클릭 간격을 추가했습니다. 실행 중 F6으로 중단할 수 있습니다.
- 비활성 창에 숫자·문자·Space·Enter·방향키와 Ctrl/Alt/Shift 조합의 실제 키다운·키업 메시지를 보냅니다.
- Windows가 변환한 문자와 별도 문자 메시지를 중복 전송하지 않아 숫자가 두 번 입력되지 않습니다.
- 키 감지에서 Space는 빈칸 대신 `{SPACE}`로 저장되고, Interception도 `{ENTER}` 같은 구문을 실제 특수키로 변환합니다.

## 작업 이름과 공개 링크

- 실행 강조 `▶`와 폴더 표시 `📁`는 화면에만 표시합니다. 기존 프로젝트에 남은 반복 접두사도 불러올 때 정리합니다.
- 프로그램의 `GitHub` 버튼은 설치 파일과 공개 문서가 있는 [YoloMacro-Distribution](https://github.com/ko9ma7/YoloMacro-Distribution)을 엽니다.

## 호환성과 검증

- 기존 프로젝트의 마우스 동작 enum 번호를 유지하므로 1.4.1 이하 프로젝트도 그대로 불러옵니다.
- 최소 활성화 호환 버전은 1.4.0을 유지하고 최신 전체 패키지는 1.4.2로 안내합니다.
- Release 빌드, 실제 Windows 비활성 숫자 입력, 2,000회 고속 클릭, 폴더 순차 실행, 보안 업데이트 11개 시나리오와 비전 품질 회귀를 검증했습니다.
